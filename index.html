About

About This Project

A senior project solving real file management pain with local AI on a Raspberry Pi NAS.

NAS Agent is an AI-powered file management system built on a Raspberry Pi 4. When you upload a file, AI automatically classifies it, generates tags, recommends a folder, and moves it — all within your local network. Find any file later with natural language search, without needing to remember exact filenames.
